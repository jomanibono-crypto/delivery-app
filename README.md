# GlovoMate

Real-time GPS sharing app for small groups with proximity notifications.

## Getting Started

1. Install Flutter SDK (see `setup_flutter.ps1` for automated install on Windows).
2. Run `flutter pub get` to install dependencies.
3. Run `flutter build apk --release` to build a release APK.

## Features

- Real-time location sharing via Firebase Realtime Database
- Anonymous authentication (no signup required)
- OpenStreetMap integration via `flutter_map`
- Proximity notifications via `flutter_local_notifications`
